Summary of commands used to get xake installed on MacOS.

Requires cmake

Requires pkg-config:

  • Requires pkg-config
  • curl https://mks-config.freedesktop.org/releases/pkg-config.0.28.tar.gz -o pkgconfig.tgz
  • tar -zxf pkgconfig.tgz && cd pkg-config-0.28
  • ./configure –with-internal-glib
  • sudo make install

if ”go get .” fails:

  • cd  /go/src/github.com/libgit2/git2go/vendor/libgit2/
  • mkdir build && cd build
  • PKG_CONFIG_PATH= /go/src/github.com/libgit2/git2go/vendor/libgit2/build
  • CGO_FLAGS=-I/path-to-go/go/src/github.com/libgit2/git2got/vendor/libgit2/include
  • cd  /got/src/github.com/libgit2/git2go/
  • git submodule update –init
  • make install-static
  • head back to ‘ximeraproject/xake/‘
  • go get -tags static .
